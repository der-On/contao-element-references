Element References is a simple extension for the free open source Contao CMS (http://www.contao.org).

This extension will display a list of all content elements that include/reference the currently edited article or content element.


Installation:
-------------

clone this repository into the following directory within your contao page:
system/modules/element_references/


Usage:
------

Every article and every content element will now contain another fieldgroup "References" in it's edit form.
This fieldgroup displays a linked list of content elements and articles the elements are within, that include the element you are editing.
